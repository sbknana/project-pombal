# Code Reviewer Agent Skills

Skills for the Project Pombal code-reviewer agent role.

## Skills

| Skill | Purpose |
|-------|---------|
| `architecture-review` | Systematic architectural assessment with SOLID, coupling, and anti-pattern detection |
| `change-impact-analysis` | Blast radius assessment, breaking change detection, dependency impact |

## Usage

Loaded automatically by the orchestrator when dispatching code-reviewer agents.
